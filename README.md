# R4MZ Quote Generator

A beginner JavaScript project built as part of my software development journey.

## Overview

The R4MZ Quote Generator is a console-based JavaScript application that helps calculate quotations for clients based on:

* Service type
* Number of hours booked
* Travel requirements
* Discount eligibility

## Features

* Photography quotations
* Videography quotations
* Photography + Videography quotations
* Automatic travel fee calculation
* Automatic discount calculation
* Console-based output

## Technologies Used

* JavaScript
* Git
* GitHub

## How It Works

1. Enter the client's name.
2. Select a service:

   * Photography
   * Videography
   * Both
3. Enter the number of hours required.
4. Specify whether travel is required.
5. The application calculates:

   * Subtotal
   * Travel fee
   * Discount
   * Final total
